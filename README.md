# Sonder's Blog

> 记录学习过程、实践复盘，以及那些值得回看的成长片段。

**在线访问**：[https://misaka789.github.io/my-blog/](https://misaka789.github.io/my-blog/)

## 功能

- 暗色/亮色主题切换
- 文章分类与站内搜索
- 阅读进度条 & 目录导航
- 代码块语法高亮、一键复制、折叠展开
- Mermaid 图表渲染
- RSS 订阅
- SEO 友好（Open Graph、Sitemap、Canonical URL）
- 响应式布局 & 移动端汉堡菜单

## 技术栈

- [Astro](https://astro.build/) — 静态站点生成
- TypeScript
- Markdown / MDX
- GitHub Pages 部署

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 本地预览构建产物
npm run preview
```
